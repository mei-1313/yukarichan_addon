# ゆかりちゃんアドオン (Yukarichan Addon)

画面の右下に常駐する和風黒髪ロングの女の子「ゆかりちゃん」が、ユーザー（マスター）のWeb閲覧に寄り添い、Gemini APIを介してお淑やかで博識なリアクションや豆知識を教えてくれるChrome拡張機能（Manifest V3）です。

---

## 🌸 主な機能

1. **インタラクション・マスコットUI**
   - すべてのWebページの右下に可愛らしく常駐します。
   - ゆかりちゃんを「つつく（クリック）」と、現在閲覧中のWebページの「タイトル」や「本文（または選択中のテキスト）」を読み込み、Gemini APIへ送信します。
   - 吹き出しのなめらかなフェードイン表示、考え中（ローディング）アニメーション、つついた時の振動などのマイクロインタラクションを実装しています。
   - 返答の内容（感情）に合わせて、ゆかりちゃんの表情（通常、照れ、喜び、悲しみ、疑問）が5パターンに変化します。

2. **Gemini API 連携**
   - 軽量・高速な `gemini-3.1-flash-lite` モデルを採用しています。
   - システム指示（System Instruction）により、丁寧でお淑やかなゆかりちゃんのキャラクター人格を固定しています。

3. **会話履歴保存**
   - 会話が行われると自動的に `chrome.storage` に「日時」「URL」「タイトル」「ゆかりちゃんのセリフ」を記録します。

4. **設定管理画面（オプション）**
   - 自身の Gemini API キーを入力・保存できます。
   - これまでのゆかりちゃんとの会話履歴をタイムスタンプ付きで綺麗にリスト表示し、必要に応じて一括クリア（削除）できます。
   - 和風モダンで美しい、ガラスマニフェズムと桜色を取り入れたデザインです。

## 📂 ディレクトリ構成

リポジトリ直下に展開されているファイル構造は以下の通りです。

```text
yukarichan_addon/
├── manifest.json         # 拡張機能のマニフェスト (Manifest V3)
├── README.md             # 本説明書
├── .gitignore            # Git除外設定 (spec.md などを除外)
├── src/
│   ├── background.js     # サービスワーカー（Gemini API通信、履歴保存、状態管理）
│   ├── content.js        # コンテントスクリプト（UI表示、クリックイベント、ページ情報取得）
│   ├── options.html      # 設定画面（APIキー入力・履歴確認用UI）
│   └── options.js        # 設定画面のロジック
└── assets/
    ├── icons/            # 拡張機能自体のアイコン (16x16, 48x48, 128x128)
    └── yukari_images/    # ゆかりちゃんの表情画像 (normal, blush, happy, sad, worry)
```

## 🛠️ インストール方法

1. このリポジトリをダウンロード、またはお使いの環境にクローンします。
2. Google Chromeブラウザを開き、アドレスバーに `chrome://extensions/` と入力して拡張機能管理画面を開きます。
3. 画面右上の **「デベロッパー モード」** を有効（ON）にします。
4. 画面左上の **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
5. 本リポジトリのルートディレクトリ（`yukarichan_addon` フォルダ）を選択して読み込みます。

---

## 💡 使い方

1. インストール後、拡張機能の一覧から「ゆかりちゃんアドオン」の **「オプション」**（設定画面）を開きます。
2. ご自身の **Gemini API キー**（`AIzaSy...`）を入力して「保存」ボタンをクリックします。
3. 任意のWebページ（例: ニュースサイトや技術ブログなど）を開くと、画面右下にゆかりちゃんが表示されます。
4. ゆかりちゃんをクリックして「つつく」と、考え中状態になった後、そのページに関連した豆知識や感想をお淑やかに語ってくれます。
5. ページ上の特定のテキストを選択した状態でつつくと、その選択範囲のテキストにフォーカスした返答をしてくれます。
